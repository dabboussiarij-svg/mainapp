How to Use Machine Status Monitoring

The machine status monitoring system displays real-time information from your Raspberry Pi-connected machines. This includes current operational status, downtime tracking, and event history.

Working

Machine is operating normally

Downtime

Machine is stopped or experiencing issues

Maintenance

Maintenance work is in progress

Break

Operator on break or lunch

Offline

Machine is powered off or disconnected
