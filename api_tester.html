Raspberry Pi Integration API Tester

Test all API endpoints for machine status monitoring

1. Get Machines List

Fetch all active machines from the system

GET /api/events/machines/list
2. Get Machine Status

Get current status of a specific machine

GET /api/events/machine_status/<machine_code>
3. Get Recent Events

Get machine events from the last N hours

GET /api/events/recent/<machine_code>?hours=24
4. Downtime Event

Start or end a downtime event

POST /api/events/downtime/<machine_code>
5. Maintenance Event

Start or end a maintenance event

POST /api/events/maintenance/<machine_code>
6. Power Cut Event

Record a power cut event

POST /api/events/power_cut/<machine_code>
How to Use
  1. First, run Get Machines List to see available machines
  2. Use a machine code from the list to test other endpoints
  3. Events are simulated - use the Start/End buttons to create event pairs
  4. All responses will appear below each button
  5. Check browser console (F12) for detailed error messages
